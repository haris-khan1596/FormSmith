# FormSmith

It is a Dynamic Form Builder. Here you can create forms 


## How to install

1. First make an virtual enviroment
```
python -m venv venv
```
2. Install Python Packages
```
pip install requirement.txt
```
3. Start Server
```
uvicorn app.main:app --reload
```
